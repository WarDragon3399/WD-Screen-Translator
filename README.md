WD Screen Translator v1.0
Developed by Wardragon3399

A lightweight, real-time screen translation tool designed for gamers and PC users. This application captures text from a specific game window using Windows OCR and translates it instantly using Google Translate.

🚀 Features
Window Specific: Only translates the game you select, allowing you to use other apps without interference.

Real-time OCR: Uses high-performance WinRT OCR for fast text recognition.

Customizable Visuals: Change font and shadow colors to ensure translations are readable against any game background.

Adjustable Scan Rate: Control how often the screen is scanned (default is 3 seconds).

Dark Mode UI: Native Windows Dark Mode support for a modern look.

📦 Installation & Setup
Download the latest release folder.

Ensure WD Screen Traslator.exe and WD.ico are in the same folder.

Run WD Screen Traslator.exe.

🛠 How to Use
Select Game: Open your game first, then click the ↻ (Refresh) button in the translator and select your game from the dropdown.

Set Languages: Choose the Source (the language in the game) and the Target (your language).

Customize: Pick your preferred colors and scan rate.

Start: Click START TRANSLATOR. The main menu will hide, and a small floating bubble will appear.

Translate: Move the floating bubble over the area you want to translate and click the Play button.

⚠️ Requirements
OS: Windows 10 (Version 1903 or later) or Windows 11.

Language Packs: For the best results, ensure the Windows Language Pack for your "Source Language" is installed in your Windows Settings.

Internet: An active internet connection is required for translations.
